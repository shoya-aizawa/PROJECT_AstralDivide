# 『Astral Divide』プロローグ・ボイス収録＆セリフ台帳（作家・キャスト用）

本書は、現行プロローグ『星の夢』（内容上の中核シーンは「星が降る丘」）におけるボイス収録およびテキスト・演出チェック用のセリフ台帳です。
ト書き（状況説明）、セリフID、キャラクターごとの演技指示（感情・声調）、およびシナリオの分岐構造をわかりやすく整理しています。
