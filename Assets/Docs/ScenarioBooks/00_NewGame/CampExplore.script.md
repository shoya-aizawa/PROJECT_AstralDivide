# Camp Explore

Render module: `Src/Stories/Scenes/00_NewGame/CampExplore.bat`

疎開キャンプの任意探索パート。  
`WASD` で注目地点を移動し、`E` で調べ、`Q` で終了して丘へ向かう。  
どの地点もフラグには直結せず、世界観と戦時下の空気を補強するための寄り道として配置する。

## Spot 1
name: カオレオン人のラジオ
text_role: 前線報告を流し、現在の戦況をぼんやり伝える。

## Spot 2
name: 配給に不満を持つ避難民
text_role: 物資不足と避難生活の疲弊を見せる。

## Spot 3
name: 看病する母親
text_role: 病人や子どもがこの場に集められている理由を感覚的に伝える。

## Spot 4
name: 焚き火の輪
text_role: 丘の噂を自然に補強し、次の本筋へ導く。

## Spot 5
name: 両国民の噂話
text_role: 同じキャンプに異なる国の避難民が混ざっている緊張感を出す。

## Spot 6
name: 見回り兵
text_role: 今は静かでも前線が完全に遠のいたわけではないことを示す。
